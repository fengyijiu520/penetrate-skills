# Java认证授权审计
扫描Java项目源码，检测权限绕过、IDOR、认证缺失漏洞，覆盖各类权限控制框架。
---
## 核心检测规则
### 1. 认证缺失检测
检测敏感接口是否缺失@PreAuthorize等认证注解。
### 2. 授权缺失检测
检测敏感接口是否缺失权限验证，是否存在越权风险。
### 3. IDOR漏洞检测
检测资源所有权验证缺失，是否存在水平越权风险。
### 4. Shiro权限绕过检测
检测Shiro框架的URL匹配绕过漏洞。
### 5. Spring Security权限绕过检测
检测Spring Security的路径匹配、AntPathMatcher绕过风险。
### 6. 注解权限检测
检测@RequiresPermissions、@Secured等注解的权限验证风险。
### 7. JWT权限检测
检测JWT token的验证缺失、签名绕过风险。
### 8. 反编译审计
支持对Jar包中的class文件进行反编译，审计字节码中的权限风险。
### 9. 误报过滤
通过执行条件分析，过滤误报。
---
## 工作流程
1. 识别敏感操作接口，定位用户输入源
2. 检测认证、授权、资源所有权的验证逻辑
3. 验证是否存在权限绕过风险
4. 分析执行条件，过滤误报
5. 生成审计报告
