# Java不安全配置审计
扫描Java项目的配置文件，检测不安全的配置，如安全配置缺失、危险配置等。
---
## 核心检测规则
### 1. Spring配置检测
检测Spring的安全配置，如csrf禁用、debug开启等风险。
### 2. Shiro配置检测
检测Shiro的rememberMe加密密钥硬编码、权限配置风险。
### 3. 安全头配置检测
检测HTTP安全头的配置缺失，如HSTS、CSP等。
### 4. 会话配置检测
检测会话的超时配置、固定会话攻击风险。
### 5. 日志配置检测
检测日志的敏感信息泄露风险。
### 6. 误报过滤
检测配置的上下文，过滤安全场景。
---
## 工作流程
1. 识别项目的配置文件
2. 检测配置项的风险
3. 验证是否存在不安全的配置
4. 过滤误报
5. 生成审计报告
