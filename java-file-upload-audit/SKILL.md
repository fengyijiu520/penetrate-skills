# Java文件上传审计
扫描Java项目源码，检测任意文件上传、文件类型绕过漏洞，覆盖文件上传接口的各类风险场景。
---
## 核心检测规则
### 1. 文件类型验证检测
检测是否存在后缀白名单验证，是否存在绕过风险。
### 2. 文件路径验证检测
检测上传文件的路径是否存在用户可控输入，路径遍历风险。
### 3. 文件内容验证检测
检测是否存在文件Magic头验证，防止文件类型绕过。
### 4. 文件名处理检测
检测getOriginalFilename的处理，是否存在路径拼接风险。
### 5. 存储权限检测
检测上传文件的存储目录是否有执行权限，防止WebShell执行。
### 6. 反编译审计
支持对Jar包中的class文件进行反编译，审计字节码中的文件上传风险。
### 7. 误报过滤
通过执行条件分析，过滤误报。
---
## 工作流程
1. 识别文件上传接口，定位MultipartFile参数
2. 检测文件类型、路径、内容的验证逻辑
3. 验证是否存在绕过风险
4. 分析执行条件，过滤误报
5. 生成审计报告
