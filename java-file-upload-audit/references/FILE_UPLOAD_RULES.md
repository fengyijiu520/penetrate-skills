# 文件上传检测规则
整合了多来源的检测规则，去重后共7种专项检测方式，覆盖任意文件上传、文件类型绕过场景。
---
## 文件上传接口检测
```java
// 文件上传Sink
MultipartFile.getOriginalFilename()
MultipartFile.transferTo(file)
FileUtils.copyFile(src, dest)
```
检测规则：
1. 检测文件上传接口的参数风险
---
## 文件类型验证检测
检测规则：
1. 检测是否存在后缀白名单验证
2. 检测是否存在Magic头验证
3. 检测是否存在文件类型绕过风险
---
## 文件路径验证检测
检测规则：
1. 检测上传路径是否存在用户可控输入
2. 检测是否存在路径遍历风险
---
## 存储权限检测
检测规则：
1. 检测上传目录是否有执行权限
2. 检测是否存在Web目录上传风险
---
## 误报过滤规则
1. 检测是否存在严格的验证逻辑
2. 检测执行条件，过滤安全场景
