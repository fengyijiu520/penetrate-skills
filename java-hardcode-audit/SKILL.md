# Java硬编码凭据审计
扫描Java项目源码，检测硬编码的敏感凭据，如密码、密钥、API Key、数据库凭据等。
---
## 核心检测规则
### 1. 数据库凭据检测
检测硬编码的数据库用户名、密码。
### 2. API Key检测
检测硬编码的API Key、Access Key、Secret Key。
### 3. 加密密钥检测
检测硬编码的加密密钥、AES密钥、RSA密钥。
### 4. 密码检测
检测硬编码的用户密码、管理员密码。
### 5. 误报过滤
检测是否是测试凭据，过滤误报。
---
## 工作流程
1. 扫描源码中的字符串常量
2. 识别敏感凭据的模式
3. 验证是否是硬编码的敏感信息
4. 过滤测试凭据等误报
5. 生成审计报告
