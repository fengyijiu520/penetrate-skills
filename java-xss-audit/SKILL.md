# Java XSS审计
扫描Java项目源码，检测跨站脚本漏洞，覆盖各类输出场景的风险。
---
## 核心检测规则
### 1. Servlet输出检测
检测response.getWriter().write的输出是否存在用户可控输入，未转义。
### 2. JSP输出检测
检测JSP中的out.print的输出，未转义的风险。
### 3. 模板引擎检测
检测Velocity、FreeMarker、Thymeleaf等模板引擎的未转义输出。
### 4. JSON输出检测
检测JSON响应中的用户输入未转义的风险。
### 5. 富文本过滤检测
检测富文本的XSS过滤是否存在绕过风险。
### 6. 误报过滤
检测是否存在HTML转义，过滤安全场景。
---
## 工作流程
1. 识别输出接口，定位用户输入源
2. 检测输出是否存在用户可控输入
3. 验证是否存在HTML转义
4. 分析执行条件，过滤误报
5. 生成审计报告
